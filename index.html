Fonctionnement

Comment ca marche ?

C'est tout simple. Tu n'as rien a faire. Le systeme tourne tout seul, toutes les 5 minutes.

1

Un contrat est signe sur MyNotary

Des qu'un mandat de vente, un mandat de recherche ou une offre d'achat est signe, le systeme le detecte automatiquement.

2

Verification Efficiale

Le systeme verifie chaque personne sur les listes officielles : Gel des avoirs (sanctions) et PPE (personnes politiquement exposees). Pour les societes, on verifie aussi les beneficiaires effectifs.

3

Le rapport PDF est genere

Un rapport de conformite est cree pour chaque personne verifiee, avec tous les details : identite, resultat des recherches, alertes eventuelles.

4

Depot dans MyNotary

Le rapport est uploade dans l'espace de stockage du dossier MyNotary. Tu le retrouves directement dans le dossier, sans rien faire.
